# MUSHBORG 
Grow mushroom easy

## Mushborg è:

1. Un applicativo di gestione del processo di lavorazione della produzione di funghi in ogni suo aspetto. Dalla creazione di nuovi “inoculi” e di spawn fino alla raccolta, conservando la storia del singolo elemento per una migliore gestione della “coltivazione”
2. Uno strumento per organizzare le fasi produttive con calendario, diario e un elenco delle cose da fare
3. Un applicativo di telemetria e controllo flessibile dell’ambiente di produzione che permette di acquisire dati e mantenere la storia di produzione
